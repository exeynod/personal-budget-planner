# Sketch 013 — Управление (top-level tab)

## Design Question
Как выглядит таб «Управление» — список пунктов: Подписки / Шаблон / Категории / Настройки.

## Variants
- **A:** Меню с описаниями — каждый пункт раскрывает контекст («3 активные · 1 097 ₽/мес»). Дороже визуально, но информативно
- **B:** С счётчиками-бейджами справа — minimalist counters (3 / 14 / 14)
- **C:** Compact list — только иконка + название, без описаний и бейджей

## Background
В v0.3 «Ещё» переименовывается в «Управление» с тем же содержимым (сейчас `MoreScreen` содержит ровно эти 4 пункта). Это не свалка — это раздел «настроек системы», ясно отделённый от продуктовых табов.

## Style
- Top-level: PageTitle с h1
- Меню-пункт: surface card с иконкой 36×36, title + опц. desc, chevron `›`
- Иконки: Phosphor line-style (Bell, FileText, Tag, Gear)
- Без саб-секций сначала; «О приложении» опционально внизу

## Winner
**A: меню с описаниями** — per user feedback, простой список с контекстной строкой даёт нужную плотность информации без перегруза. Совпадает с текущей реализацией `MoreScreen`.
