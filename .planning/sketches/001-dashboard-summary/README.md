# Sketch 001: Dashboard Summary

## Design Question
Главный экран Mini App — эквивалент листа Summary. Должен в 375px помещать:
переключатель периода, hero-баланс, прогноз, агрегаты по расходам/доходам и
14 категорий с план/факт/дельтой. Какая структура удобнее: collapsible секции,
табы или единый список с heading-чипами?

## How to View
```
open .planning/sketches/001-dashboard-summary/index.html
```

## Variants
- **A: Stacked + collapsible** — Hero, ниже две сворачиваемые секции «Расходы» / «Доходы». Карточки с прогресс-барами факт/план и дельтой в строке. Активный пользовательский flow: расходы открыты, доходы свёрнуты по умолчанию.
- **B: Tabs Расходы / Доходы** — Hero, под ним сегментированный таб-свитчер (с counter-чипами «11» / «3»). Каждая вкладка содержит aggr-карточку с тремя цифрами (План/Факт/Δ) и плотный список категорий. Больше контента в видимой зоне.
- **C: Summary cards + единый список** — Hero, ниже две side-by-side карточки-агрегата (стек-бар + дельта), затем единый плотный список категорий с цветными чипами «расход/доход» и разделителями. Самая «банковская» подача: компактно, минимум переходов, всё в один скролл.

## What to Look For
- **Скорость поиска нужной категории** — насколько быстро глазом находится «Дом» или «Аванс».
- **Соотношение информации/воздуха** — где плотнее, где переборщили с paddings.
- **Hero-фокус** — не перетягивает ли agg-блок (B/C) внимание с баланса.
- **Прогрессы-бары при пустом факте** (как в этом моке) — все три варианта показывают «всё ноль» — какой при этом не выглядит мёртвым.
- **MainButton дотягиваемость** — нижняя кнопка «Добавить трату» при scroll-разной длине.
- **Переключатель Расходы↔Доходы** в B vs C: вкладки vs прокрутка.
- **Dark / light** — toggle в правом нижнем sketch-toolbar.

## Notes
- 375px iPhone viewport симулируется обрамлением `.frame`.
- MainButton эмулирован абсолютной кнопкой внизу (в реальном TG WebApp это `tg.MainButton`).
- Tap-handlers подсвечивают взаимодействие через toast.
- Цвет дельты: положительная = зелёный, отрицательная = красный (договорённость BRD §6).
- Светлая тема — переключатель в правом нижнем углу.
