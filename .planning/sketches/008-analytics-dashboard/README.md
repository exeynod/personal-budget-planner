# Sketch 008: Analytics Dashboard

## Design Question
Какой паттерн отображения аналитики читается лучше всего на мобильном экране?

## How to View
open .planning/sketches/008-analytics-dashboard/index.html

## Variants
- **A: Line chart + список категорий** — SVG-график тренда + горизонтальные бары топ-5
- **B: Метрики + бары** — карточки с ключевыми числами + интерактивный список
- **C: Hero сравнение + прогноз** — сравнение месяцев + прогноз остатка + trend line

## What to Look For
- Читается ли SVG line chart на тёмном фоне?
- Какой вариант даёт больше полезной информации с первого взгляда?
- Не перегружен ли экран?
