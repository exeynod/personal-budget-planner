# Sketch 007: Bottom Nav

## Design Question
Как разместить 5 вкладок (Главная/Добавить/Аналитика/ИИ/Ещё) с визуальным отличием AI-вкладки?

## How to View
open .planning/sketches/007-bottom-nav/index.html

## Variants
- **A: Иконки + лейблы** — стандартный mobile nav, dot-indicator под активной вкладкой
- **B: Иконки + бар** — иконки без подписей, accent-bar снизу, badge для уведомлений ИИ
- **C: Pill-nav** — все вкладки внутри pill-контейнера, pill-highlight для активной

## What to Look For
- Достаточно ли понятна AI-вкладка без лейбла?
- Не перегружает ли 5 вкладок 375px?
- Выделяется ли AI (фиолетовый) относительно остальных (синий)?
