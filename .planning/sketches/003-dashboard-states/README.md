# Sketch 003: Dashboard States

## Design Question
Sketch 001 показал дашборд в одном пустом состоянии (новый месяц, факт=0).
Реальные пользовательские состояния — другие. Этот скетч — не выбор «лучшего
варианта», а валидация четырёх состояний главного экрана.

## How to View
```
open .planning/sketches/003-dashboard-states/index.html
```

## States (вместо variants)
- **Empty** — период создан, но шаблон не применён. Hero показывает только баланс. На месте план/факт-блока — empty-state с кнопками «Применить шаблон» / «Добавить вручную».
- **In progress** — середина периода. Hero показывает «Прошло 11 / 28 дней». Категории с разной заполненностью: 45%, 37%, 87% (warn — жёлтая обводка), 6%, 0%. Дельты везде положительные (расход в рамках плана).
- **Overspend** — конец периода. Hero на красном градиенте, alert-блок «3 категории превысили». 3 категории с красной обводкой и `+212% / +34% / +8%` бейджами, прогресс-бары забиты на 100% danger. Aggr-блок дельта `−71 600` красным.
- **Closed** — read-only архивный период. Бейдж «Закрыт» в свитчере, информер сверху. Hero на синем градиенте показывает «Итоговый баланс» и «+ Накоплено». MainButton задизейблен с надписью «Период закрыт».

## Design Decisions Made
- **Прогресс ≥80%** — жёлтый warn (border + fill), бейдж процента в названии.
- **Прогресс >100%** — красный danger (border + bar забит на 100%), бейдж `+N%`, сумма факта красным.
- **Hero меняет градиент по состоянию:** обычный синий-фиолетовый, тревожный красный (overspend), приглушённый сине-зелёный (closed).
- **Информер о закрытии** — отдельный banner над hero, не часть hero.
- **MainButton дизейблится** в closed-state, текст меняется.
- **Прогноз `— ₽`** в empty (нет плана = нет прогноза).
- **Hero alert** — только в overspend, не каждый период.

## What to Look For
- Перепрыгивайте между состояниями через переключатель сверху или dropdown в правом нижнем углу.
- Цветовая логика: где зелёный, где жёлтый, где красный. Согласуется ли с правилом BRD §6 «положительная дельта = хорошо».
- Empty-state: достаточно ли понятна инструкция к действию.
- Closed: легко ли понять, что период архивный (бейдж + banner + дизейбл кнопки).
- Overspend: не пугает ли красный hero.
