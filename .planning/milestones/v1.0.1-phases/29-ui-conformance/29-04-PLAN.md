    **Iteration loop (one BLOCKER at a time):**

    1. **Pick next BLOCKER** из in-scope list (per checkpoint resume-signal).
    2. **Locate exact file:line** из finding (`File: ...` field в UI-REVIEW.md).
    3. **Apply minimal fix:**
       - **Color hex mismatch** → edit `tokens.css` value OR `PosterTokens.swift` value. Если override в module.css / SwiftUI inline literal — исправить там.
       - **Spacing/sizing** → edit module.css / SwiftUI .padding() / PosterTokens.spacing*.

       - **Wrong layout order** → reorder JSX/SwiftUI children.
