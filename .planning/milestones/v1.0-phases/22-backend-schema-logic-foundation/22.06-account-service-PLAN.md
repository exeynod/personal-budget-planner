**Phase 2: Implement `app/services/accounts.py`.**

Module structure:

1. **Domain exceptions:**
```python
class AccountNotFoundError(Exception):
    def __init__(self, account_id: int) -> None:
        self.account_id = account_id
        super().__init__(f"Account {account_id} not found")

class AccountHasTxnsError(Exception):
    """Raised when trying to delete an account with referencing actual_transactions or subscriptions (BE-02)."""
    def __init__(self, account_id: int, txn_count: int, sub_count: int) -> None:
        self.account_id = account_id
        self.txn_count = txn_count
        self.sub_count = sub_count
        super().__init__(
            f"Account {account_id} has {txn_count} transactions and {sub_count} subscriptions; "
            "delete or migrate them first"
        )

class PrimaryAccountConflictError(Exception):
    """Raised on race when partial unique index would violate."""
```

2. **`async def list_accounts(db, *, user_id) -> list[Account]`:** select all where user_id == user_id, order by is_primary DESC, created_at ASC.

3. **`async def get_or_404(db, account_id, *, user_id) -> Account`:** scoped by user_id; raise AccountNotFoundError if not found.

4. **`async def create_account(db, *, user_id, bank, kind, balance_cents, mask=None, primary=False) -> Account`:**
   - Within a single transaction (the caller's session — flush only, do NOT commit):
     - If `primary=True` OR no existing account exists for user, demote any existing primary: `UPDATE account SET "primary" = false WHERE user_id = :uid AND "primary" = true`.
     - INSERT new account with is_primary = (primary OR no_existing).
     - Flush, refresh, return.
   - Pseudocode:
     ```python
     existing = await db.scalar(select(func.count()).select_from(Account).where(Account.user_id == user_id))
     auto_primary = existing == 0
     should_be_primary = primary or auto_primary
     if should_be_primary:
         await db.execute(
             text('UPDATE account SET "primary" = false WHERE user_id = :uid AND "primary" = true'),
             {"uid": user_id}
         )
     row = Account(
         user_id=user_id, bank=bank, kind=kind,
         balance_cents=balance_cents, mask=mask,
         is_primary=should_be_primary,
     )
     db.add(row)
     await db.flush()
     await db.refresh(row)
     return row
     ```

5. **`async def update_account(db, account_id, *, user_id, patch: dict) -> Account`:**
   - Fetch via get_or_404.
   - If `patch.get("primary")` is True and currently is_primary=False: demote others first.
   - If `patch.get("primary")` is False and this is the only primary: refuse (raise ValueError "Cannot demote sole primary account").
   - Apply patch fields. Flush. Refresh. Return.

7. **`async def apply_balance_delta(db, account_id, *, user_id, delta_cents) -> int`:**
   - Atomic UPDATE returning new balance:
     ```python
     result = await db.execute(
         text(
             'UPDATE account SET balance_cents = balance_cents + :delta '
             'WHERE id = :id AND user_id = :uid '
             'RETURNING balance_cents'
         ),
         {"delta": delta_cents, "id": account_id, "uid": user_id}
     )
     row = result.first()
     if row is None:
         raise AccountNotFoundError(account_id)
     return row.balance_cents
     ```

8. **`async def get_primary_account(db, *, user_id) -> Account | None`:** select by is_primary=True, user_id=user_id, limit 1.
