⚠️ КРИТИЧНО (Networking/APIClient.swift rawRequest строки 148-157):
```swift
case 401: onUnauthenticated?(); throw APIError.unauthorized
case 403:
    let detail = decodeErrorDetail(data) ?? "Forbidden"
    if !skipAuth { onUnauthenticated?() }   // ← вызывает глобальный logout!

```
AuthStore.onUnauthenticated (Auth/AuthStore.swift:17) → handleUnauthenticated() = re-auth/logout.

APIError (Networking/APIError.swift): .forbidden(String), .unauthorized, .notFound, .network(Error), .serverError(Int,String) — все ловятся в wrapper и → nil.

    - `private(set) var suggestion: SuggestCategoryDTO? = nil`
    - injectable seam: `private let suggest: (String) async -> SuggestCategoryDTO?` через init с default `= { await AISuggestCategoryAPI.suggest(q: $0) }` (в тестах подменяется стабом).
    - конфиг через init: `minChars: Int = 3`, `debounce: Duration = .milliseconds(500)` (в тестах .zero).
    - `private var task: Task<Void, Never>?`
    - `func descriptionChanged(_ text: String)`:
        ```swift
        task?.cancel()                       // отмена предыдущего запроса
        let q = text.trimmingCharacters(in: .whitespacesAndNewlines)
        guard q.count >= minChars else { suggestion = nil; return }
        task = Task { [weak self] in
            guard let self else { return }
            try? await Task.sleep(for: self.debounce)
            if Task.isCancelled { return }
            let result = await self.suggest(q)
            if Task.isCancelled { return }    // не перетирать новым вводом
            self.suggestion = result          // nil = silent hide
        }
        ```
      (Task.isCancelled проверяется ПОСЛЕ await suggest — гарантия что медленный старый ответ не перетрёт новый.)
    - `func clear()`: `task?.cancel(); suggestion = nil`
