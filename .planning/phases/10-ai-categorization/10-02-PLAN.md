# Plan 02: Embedding service + suggest API

## Objective
Реализовать embed() метод в LLM провайдере, сервис embeddings с cosine similarity поиском, и API endpoint suggest-category.

## Tasks

### Task 1: embed() метод в AbstractLLMClient + OpenAIProvider
- Добавить abstract async embed(text: str) -> list[float] в AbstractLLMClient
- Реализовать в OpenAIProvider через text-embedding-3-small

### Task 2: Embedding service
- Создать app/services/embedding_service.py с функциями:
  - get_category_embeddings(db)
  - generate_embedding(text, llm_client)
  - upsert_category_embedding(db, category_id, vector)
  - suggest_category(db, description, llm_client) -> dict | None
  - Cosine similarity: 1 - (embedding <=> query_vec), threshold 0.5

### Task 3: AI categorization route
- Создать app/api/routes/ai_categorization.py
- GET /api/v1/ai/suggest-category?q=<text>
- Проверяет settings.enable_ai_categorization
