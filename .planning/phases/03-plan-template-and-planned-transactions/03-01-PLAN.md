**Use direct DB-insert helpers** для setup BudgetPeriod / PlanTemplateItem / PlannedTransaction / Subscription — не пытайтесь делать это через future-non-existent endpoints. Пример:
```python
async def _create_period(SessionLocal, *, period_start, period_end, starting_balance_cents=0):
    async with SessionLocal() as session:
        period = BudgetPeriod(
            period_start=period_start, period_end=period_end,
            starting_balance_cents=starting_balance_cents, status=PeriodStatus.active,
        )
        session.add(period)
        await session.commit()
        await session.refresh(period)
        return period.id
```
