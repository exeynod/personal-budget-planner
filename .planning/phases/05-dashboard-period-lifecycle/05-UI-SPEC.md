## Typography

All sizes from `--text-*` tokens in `default.css`. Phase 5 uses exactly 4 sizes and exactly 2 weights:

| Role | Token | Size | Weight | Line Height | Usage |
|------|-------|------|--------|-------------|-------|
| Body | --text-sm | 13px | 400 (--weight-regular) | 1.5 | Category name, description, muted labels, section overlines, overspend badge |
| Label | --text-base | 15px | 400 (--weight-regular) | 1.5 | Aggr-block values, period switcher text |
| Heading | --text-md | 17px | 600 (--weight-semibold) | 1.2 | Category row amount, aggr-block amount |
| Display | --text-2xl | 32px | 600 (--weight-semibold) | 1.1 | Hero card balance_now_cents |

Mono font (`--font-mono`) applies to all currency amounts — hero balance, aggr totals, category planned/actual values, delta badge.

Supplementary:
- Period switcher month label: --text-base (15px), weight 600 (--weight-semibold), centered
- Section overline (e.g. "РАСХОДЫ"): --text-sm (13px), weight 600, letter-spacing 0.06em, uppercase
- Progress percentage badge (overspend): --text-sm (13px), weight 600

## Color

Source: `default.css` tokens. No new colors introduced in Phase 5.

| Role | Token | Value | Usage |
|------|-------|-------|-------|
| Dominant (60%) | --color-bg | #0e1116 | Screen background, FAB shadow base |
| Secondary (30%) | --color-surface / --color-surface-2 | #1c2230 / #232a3a | Category rows, aggr strip, period switcher bar |
| Accent (10%) | --color-primary | #4ea4ff | Tab active underline, period nav arrows, progress bar (normal state), primary CTA buttons |
| Success | --color-success | #2ecc71 | Positive delta values (Plan−Fact > 0 for expenses; Fact−Plan > 0 for income) |
| Danger | --color-danger | #ff5d5d | Negative delta values; overspend progress bar; overspend border; overspend badge; destructive actions |
| Warning | --color-warn | #ffb547 | Warn border (≥80% consumed); warn progress bar |
| Hero gradient | --gradient-hero | linear-gradient(135deg, #1d2740 0%, #2d1d40 100%) | Hero card background only |
| Hero glow | --gradient-hero-glow | radial-gradient(circle at 20% 0%, rgba(78,164,255,0.25), transparent 60%) | Hero card overlay glow only |

Accent (--color-primary) reserved for:
- Active tab underline/indicator
- Period switcher navigation arrows (‹ ›)
- Normal-state progress bar fill
- "Применить шаблон" CTA button background (empty state)
- Overspend badge uses --color-danger, not accent

Semantic color rules:
- Delta display: positive value → --color-success text; negative value → --color-danger text; zero → --color-text-muted
- Closed period badge: --color-text-muted text on --color-surface-2 background (neutral, not danger)
- Warn category row: 1px solid --color-warn border, --color-warn progress fill
- Overspend category row: 1px solid --color-danger border, --color-danger progress fill, --color-danger badge

## Component Inventory

### New in Phase 5

**HeroCard** (`components/HeroCard.tsx`)
- Background: `--gradient-hero` + `--gradient-hero-glow` overlay
- Border-radius: `--radius-lg` (20px)
- Padding: `--space-6` (24px) horizontal, `--space-5` (20px) vertical
- Shadow: `--shadow-md`
- Content layout: 3 rows stacked
  - Row 1: Period label — format "5 апр – 4 мая 2026" — --text-sm, --color-text-muted
  - Row 2: Balance amount — `formatKopecks(balance_now_cents)` — --text-2xl, --weight-semibold, --font-mono; on closed period use `ending_balance_cents` with label "Итог"
  - Row 3: Delta — `formatKopecksWithSign(delta_total_cents)` — --text-base, --font-mono; color determined by sign and kind rule; label "Δ"

**PeriodSwitcher** (`components/PeriodSwitcher.tsx`)
- Layout: single horizontal row, height 36px
- Structure: `[‹]  [Май 2026  [Закрыт]]  [›]`
- Arrows: Unicode "‹" / "›", --color-primary; disabled state → --color-text-dim, pointer-events none
- Month label: --text-base, --weight-semibold, centered
- "Закрыт" badge: pill shape, --radius-full, padding 2px 8px, --text-sm, --color-text-muted text, --color-surface-2 background — appears only when period status = closed
- No badge on active period
- Margin-bottom: --space-3 (12px) above tabs

**TabBar** (`components/TabBar.tsx` or inline in HomeScreen)
- Two tabs: "Расходы" / "Доходы"
- Height: 44px
- Active tab: --color-text, --weight-semibold, 2px solid --color-primary underline
- Inactive tab: --color-text-muted, --weight-regular, no underline
- Background: --color-bg-elevated (#161a22)
- Sticky: position sticky, top 0, z-index 20
- No border-radius on tabs themselves

**AggrStrip** (inline section or `components/AggrStrip.tsx`)
- Fixed strip directly below TabBar, above category list
- Background: --color-surface (#1c2230)
- Border-bottom: 1px solid --color-border-subtle
- Padding: --space-3 (12px) vertical, --space-4 (16px) horizontal
- Layout: 3 equal columns — "План", "Факт", "Δ"
- Each column: label --text-sm uppercase --color-text-dim, value --text-md --font-mono --weight-semibold
- Delta value color: sign-based (--color-success / --color-danger / --color-text-muted)

**DashboardCategoryRow** (extends/replaces CategoryRow for dashboard use)
- Height: auto (min 52px for touch)
- Layout: left=category name; right=actual / planned amounts + progress bar below
- Category name: --text-base, --weight-regular, --color-text; truncate with ellipsis
- Amounts: --text-sm, --font-mono; actual in --color-text, planned in --color-text-muted; format "4 200 / 10 000 ₽"
- Progress bar: full width, height 4px, border-radius 2px; background --color-border; fill width = min(actual/planned × 100%, 100%); no bar when planned = 0
- Progress fill colors:
  - < 80%: --color-primary
  - ≥ 80% and ≤ 100%: --color-warn
  - > 100%: --color-danger
- Warn border: 1px solid --color-warn on row container
- Overspend: 1px solid --color-danger border + overspend percentage badge
- Overspend badge: pill, --color-danger background (--color-danger-soft), --color-danger text, --text-sm, --weight-600, shows "123%"
- Zero-actual rows: show row normally with 0 actual, empty progress bar (no bar if planned = 0)
- Row padding: --space-3 (12px) vertical, --space-4 (16px) horizontal
- Row background: --color-surface; hover --color-surface-2
- Row border-radius: --radius-md (14px)
- Row margin-bottom: --space-2 (8px)

**EmptyStateDashboard** (inline in HomeScreen)
- Centered vertically in category list area
- Icon: Unicode budget icon or simple SVG (e.g. 📋 or inline clipboard SVG)
- Heading: --text-md, --weight-semibold, "Бюджет не запланирован"
- Body: --text-sm, --color-text-muted, "Примени шаблон или добавь строки вручную"
- CTA row: two buttons stacked or side-by-side with gap --space-3
  - Primary: "Применить шаблон" — background --color-primary, white text, --radius-md, height 44px, full width
  - Secondary: "Добавить вручную" — border 1px --color-border, transparent background, --color-text, --radius-md, height 44px, full width

## Screen Layout Contract

### HomeScreen (full replacement)

Viewport: 375px max-width, centered, background --color-bg.

```
┌─────────────────────────────────┐  ← screen top
│  [HeroCard]                     │  ← padding-top --space-4; card full-width
│  [PeriodSwitcher]               │  ← margin-top --space-4
│  [TabBar]  (sticky)             │  ← sticks to top on scroll
│  [AggrStrip]                    │  ← below tabs, not sticky
│  ─────────────────────────────  │
│  [Category rows]                │  ← scrollable list
│  ...                            │
│  [EmptyState] (if no plan)      │
│                                 │
│  [FAB]  bottom-right            │  ← fixed; hidden on closed
│  [MainButton] (closed only)     │  ← fixed bottom; "Период закрыт"
└─────────────────────────────────┘
```

Screen-level padding: 0 horizontal at root, --space-4 (16px) horizontal inside content sections.
Bottom safe area: `calc(var(--safe-bottom) + var(--main-button-height) + 8px)` on closed periods to avoid MainButton overlap.

## Interaction Contracts

### Period Navigation (PeriodSwitcher)

| Action | Condition | Result |
|--------|-----------|--------|
| Tap ‹ | Previous period exists | Navigate to previous period; update all data |
| Tap ‹ | No previous period | Arrow disabled, no action |
| Tap › | Not on current active period | Navigate forward one period |
| Tap › | On current active period | Arrow disabled, no action |
| Period loaded, status=closed | Always | Show "Закрыт" badge; FAB hidden; MainButton shows "Период закрыт" (disabled) |

### Tab Switching

| Action | Result |
|--------|--------|
| Tap "Расходы" tab | Filter category list to kind=expense; AggrStrip shows expense totals |
| Tap "Доходы" tab | Filter category list to kind=income; AggrStrip shows income totals |
| Tab switch | No scroll reset; category list re-renders with filtered data |

### Empty State Actions

| Action | Result |
|--------|--------|
| Tap "Применить шаблон" | POST /periods/{id}/apply-template; show toast "Шаблон применён"; reload category data |
| Tap "Добавить вручную" | Navigate to PlannedScreen |
| Template apply fails | Show inline error below buttons: "Не удалось применить шаблон. Попробуй ещё раз." |

### FAB

| Condition | Visibility |
|-----------|-----------|
| Active period | Visible, fixed bottom-right, z-index 100 |
| Closed period | Hidden (display: none) |
| Archived period | Hidden (display: none) |
| BottomSheet open | FAB remains visible behind sheet scrim |

### Closed Period Read-Only

| Element | Behavior |
|---------|----------|
| FAB | Hidden |
| MainButton | Visible, disabled, label "Период закрыт" |
| Category rows | No click/tap action; cursor default |
| AggrStrip | Read-only display |
| Hero card balance | Shows ending_balance_cents with label "Итог периода" |

## Loading & Error States

| State | Visual Treatment |
|-------|-----------------|
| Initial load | Hero card: skeleton placeholder (--color-surface-2 animated pulse); category list: 4 skeleton rows |
| Data error | Inline message in category list area: --color-danger-soft background, "Не удалось загрузить данные. Потяни вниз для обновления." |
| Period switch loading | PeriodSwitcher arrows dimmed; category list shows skeleton; hero card shows previous data until new data arrives |
| Apply template loading | "Применить шаблон" button shows "…" label, disabled; prevent double-tap |
| Toast duration | 2200ms auto-dismiss (matches established pattern in HomeScreen and PlannedScreen) |
| Toast style | Pill shape, --radius-full, position fixed bottom 100px, centered; success toasts: --color-success-soft bg, --color-success text, --color-success border 1px |
