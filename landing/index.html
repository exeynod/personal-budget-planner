FAQ

Частые вопросы.

Чем отличается от Дзен-мани / CoinKeeper / Тинькофф?
Tinkoff Бюджет работает только с карточными расходами через MCC. Дзен-мани требует SMS-парсинг + ручной cash-ввод без AI. CoinKeeper — без conversational AI. TG Budget Planner — единственный с AI-чатом + cash + Telegram-native (без отдельного приложения).
Безопасно ли держать финансовые данные у вас?
Postgres RLS на 9 доменных таблицах — данные физически изолированы между пользователями на DB-уровне (composite FK). Согласие на обработку ПДн при первом запуске. Удаление аккаунта через `/me/account DELETE` — soft-delete 30 дней + автоматический purge. Open-core: код доступен для self-hosting.
Как платить из России?
ЮKassa Self-Employed — карта любого банка РФ, СБП. Чеки автоматически передаются в ФНС «Мой Налог». Никаких VPN.
Что значит «открытый код / open-core»?
Ядро + Telegram-бот + Docker self-host публикуются под лицензией PolyForm Shield на GitHub. Хочешь — разверни локально бесплатно. AI / iOS / multi-tenant cloud — proprietary, разлочиваются в hosted Pro версии.
А если я бросаю — мои данные пропадут?
Нет. Endpoint `GET /me/export` отдаёт ZIP со всеми твоими данными в JSON в любой момент. Account deletion = soft-delete с 30-day cooling period: одумаешься — восстановим. После 30 дней — hard purge через worker.
